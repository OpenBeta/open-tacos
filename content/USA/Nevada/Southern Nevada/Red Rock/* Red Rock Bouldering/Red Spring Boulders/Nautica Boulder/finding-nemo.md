# Description
Start on the port (left) side of the hull face and traverse to the left using obvious roof jugs. Make your way up the face upon reaching the crack, using it for a couple of layback moves before stepping up to a soft slab top-out.# Location
To the right of Weigh Anchor.# Protection
1-2 pads
