# Description
Starts up dark colored overhanging rock, then trends right onto red and cream colored, sandy face(crux area). Probably best to preclip first bolt with stick or by using back wall for stance.# Location
This is highest up route in gully, on the left edge of the cliff face.# Protection
bolts and anchors
