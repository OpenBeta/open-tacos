# Description
This route climbs an obvious right facing corner to the left of Beauty. Start in an offwidth crack, which can easily be avoided by using near by face holds.  Climb to the ledge and beware of loose rock.  Move slightly left and climb a combination of the corner/offwidth/finger crack.  Belay off a tree at top.To get back to the base lower off a tree above Beauty with a 70m.# Location
Crack/corner 20' the left of Beauty.# Protection
Single rack to 5", double 5" to avoid a run out.
