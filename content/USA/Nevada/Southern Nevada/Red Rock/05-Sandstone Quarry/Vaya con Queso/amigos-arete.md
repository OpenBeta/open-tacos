# Description
Stand start with crimps at head height (maybe a little higher), left of the obvious crimp rail start of Mexican Jacket. The left hand starting crimp, is on the actual arete. Climb the arete. Use the ledge at top to hand traverse leftwards and top out.# Location
The obvious arete between Dusty Taco and Mexican Jacket.# Protection
Bouldering pads and spotters.
