# Description
Start at a left facing flake and climb to the first bolt.  Pass two more bolts and run it out to the anchor.  It is possible to place gear between the last bolt and the anchor, but be careful.  There are some loose flakes.# Location
Elementary Primer starts about 10 or 15 feet to the left ofStone Hammer.# Protection
3 Bolts to the anchor
