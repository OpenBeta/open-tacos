# Description
straight up the seam on small, crispy holds to the ledge above. you can walk up to the ledge to set up a toprope, which is probably the best way to climb this thing.# Location
Walk all the way to the right of the Panty Wall proper, past Butt Floss et al. Around the corner and up a bit is a short, friable looking seam.# Protection
small nuts which can be placed as mostly psychological pro. bolted anchor on ledge.
