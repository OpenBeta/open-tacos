# Description
This route climbs the blunt arete situated between the two sections of the wall. Third class up and to the right (and the first bolt), then edge straight up on solid little varnish crimpers to the anchors. The 20-foot face directly beneath the climb can be toproped at the same difficulty.# Location
# Protection
Bolts to chains.
