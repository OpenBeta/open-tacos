# Description
Short but fun finger and hand crack. Start sitting with two hand jams, then make moves up over the bulge to a narrowing finger crack and some crimps. It looks like a pile but it climbs well.A fun eliminate that uses only the crack for hands goes at about v2 is also recommended.# Location
Back side of parking lot boulder. Finger and hand crack on right side.# Protection
Pad or two.
